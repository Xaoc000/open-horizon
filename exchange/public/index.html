horizon.dcmartin.com

Overview

Detecting entities of interest (e.g. person, animal, vehicle) using the YOLO object detection and classification software from network accessible and direct-attach cameras.

Required Software

  • Docker - Packaging for software delivery system as container
  • Home Assistant - Home automation
  • YOLO - You-Only-Look-Once object detection and classification AI
  • OpenHorizon - Distributed edge fabric for containers

Containers

motion

A Home Assistant addon for the motion project software

yolo4motion

A Open Horizon service for the YOLO software
